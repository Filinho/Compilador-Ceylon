
# Ceylon Compiler

This project is a college assignment for the Compilers course. The goal is to develop a simple compiler for the Ceylon language, covering the main stages of compilation.

## Project Overview
The compiler is organized in stages, each in its own folder:
- **Lab-01:** Lexical analysis (lexer/tokenizer)
- (Other labs may cover syntax analysis, semantic analysis, code generation, etc.)
